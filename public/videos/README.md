# Hero Video Assets

## Required Video: `hero-websites.mp4`

### Video Specifications
- **Location:** Place video file here as `hero-websites.mp4`
- **Resolution:** 1920x1080 (Full HD) minimum
- **Aspect Ratio:** 16:9
- **Duration:** 10-20 seconds (will loop seamlessly)
- **Format:** MP4 (H.264 codec)
- **File Size:** Under 5MB recommended (compress if needed)
- **Opacity:** Video displays at 40% opacity automatically

- **Theme:** Abstract, modern, tech-focused

- **Content Ideas:**

  - Glass morphism UI elements morphing
  - Abstract web interface animations
  - Geometric shapes/patterns
  - Neural network visualizations
